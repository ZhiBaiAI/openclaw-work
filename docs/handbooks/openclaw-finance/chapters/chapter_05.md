# 第5章：实战一 — 发票处理自动化

## 5.1 场景描述与痛点

### 真实痛点

发票处理是财务最常见的重复性劳动：

| 痛点 | 具体表现 | 影响 |
|------|---------|------|
| **效率低** | 手工录入一张发票需要3-5分钟 | 月均200张 = 10-17小时 |
| **易出错** | 人工录入错误率约2-5% | 月均4-10张需要返工 |
| **格式乱** | 纸质票/PDF/拍照/电子发票混杂 | 需要多种工具切换 |
| **汇总难** | 按月份、供应商、类型分类汇总 | 额外2-3小时 |

### 方案一：识别发票内容

把发票图片或 PDF 扔给 WorkBuddy，直接说：

```

- 购买方名称

- 金额（含税/不含税）
- 税额
- 税率

输出表格格式
```

**你只需要**：把发票截图/文件发给 WorkBuddy

```
1. 每张发票的汇总表（发票号、日期、金额、类型）
```

### 方案三：发票验真

需要查验发票真伪？直接说：

```
检查这张发票是否真实有效：
[粘贴发票信息]

方式一：对比历史数据
- 检查这张发票的金额、日期是否与报销单一致

方式二：格式检查
- 检查发票代码、号码格式是否正确
```

## 5.3 实战案例

### 案例：30分钟处理100张发票

**任务背景**：财务小李每月要处理约100张发票，整理归档到系统中。

**传统方式**：5小时（逐张录入 → 分类 → 汇总 → 归档）

**用 WorkBuddy**：

```
第一步：批量识别
"处理桌面上的'3月发票'文件夹，读取所有发票图片，
提取每张发票的金额和日期，输出汇总表"
```

```
第二步：自动分类
"根据发票内容，将这100张发票按以下规则分类：
- 机票/火车票 → 差旅费
- 办公用品、文具 → 办公费
- 餐费 → 业务招待费
- 其他 → 其他费用

输出分类汇总表"
```

```
第三步：生成归档文件
"按发票日期重新命名这些文件，
格式：'类型_YYYYMMDD_金额.pdf'，
然后整理到一个新文件夹"
```

**结果**：30分钟完成5小时的工作，准确率更高。

## 5.5 常见问题

### Q1：WorkBuddy 识别发票慢怎么办？

发票识别需要时间处理图片，大文件可能慢一些：
- 可以分批处理（每次20-30张）
- 高清图片可以先压缩一下

### Q2：发票图片不清晰识别不出来怎么办？

尽量使用：
- 电子发票 PDF（最清晰）
- 扫描件（比拍照清晰）
- 拍照时保持光线充足、图片清晰

### Q3：如何提高识别准确率？

在指令中明确发票类型：
```
这是增值税专用发票，请重点提取：发票代码、发票号码、金额、税额
```

## 5.6 更多高频财务场景

除了发票处理，这些场景也是财务日常工作的高频需求：

### 场景1：报销审核

**痛点**：每月几百张报销单，人工核对费时费力，还容易漏掉重复报销、超标准等问题。

**WorkBuddy 帮你做**：

```
审核这个月的报销单文件（Excel格式）

检查项目：
1. 重复报销：同一发票号是否出现多次
2. 超标检查：差旅费是否超过部门预算
3. 票据齐全：金额大于500元是否有发票
4. 日期逻辑：报销日期是否在发票日期之后
5. 签字完整：是否有审批人签字

输出：
- 问题清单（列出每条问题所在的行号）
- 通过的报销单数量
- 建议：哪些需要打回修改
```

**效率对比**：原来2人×2天 → 现在10分钟

---

### 场景2：工资核算辅助

**痛点**：每月工资条制作、个税计算、社保核对，工作量大又容易出错。

**WorkBuddy 帮你做**：

```
根据这份工资表（Excel）：

1. 计算每个员工的应纳税所得额
2. 计算个税（使用5000元免征额+专项附加扣除）
3. 核对社保个人扣款是否正确
4. 生成工资条（隐藏敏感信息版）

输出工资条表格，保存为"6月工资条.xlsx"
```

**提示**：首次使用需要告诉 WorkBuddy 公司的社保缴纳比例和公积金比例。

### 场景3：发票归档整理

**痛点**：发票太多太乱，按年份/部门/项目分类整理耗时。

**WorkBuddy 帮你做**：

```
整理这个文件夹（D:\财务归档\2026年）中的所有发票

归档规则：
1. 按年份→月份→类型 建立文件夹结构
2. 文件命名格式：类型_YYYYMMDD_金额
3. 生成归档清单Excel，包含：原文件名、新路径、发票号、金额

同时检查：
- 是否有重名文件
- 是否有缺失发票号的情况
```

### 场景4：银行回单处理

**痛点**：银行导出的回单PDF/Excel，散落在邮件或网银系统里，整理归档麻烦。

**WorkBuddy 帮你做**：

```
处理这个文件夹中的银行回单文件：

1. 读取每个回单，提取：
   - 交易日期
   - 对方账户名
   - 金额
   - 摘要/用途

2. 按月份汇总收支
3. 生成汇总表：月份、收入合计、支出合计、净额
4. 检查是否有大额异常交易

输出汇总表Excel
```

---

### 场景5：税务申报数据整理

**痛点**：申报前要把分散的数据整理成申报表格式，容易填错。

**WorkBuddy 帮你做**：

```
整理这份销售数据（Excel），为增值税申报做准备：

1. 按税率分类汇总销售额和销项税额
2. 核对：不含税金额 × 税率 = 税额 是否成立
3. 找出金额异常的月份

输出：
- 分类汇总表
- 异常月份清单
- 申报表填写建议值
```

**⚠️ 注意**：WorkBuddy 帮你整理数据，但申报前务必人工核对确认。
