## 6.1 场景描述与痛点

### 财务数据分析的现状

> "80%的时间都在整理数据，真正做分析只有20%——这大概就是财务数据搬运工的真实写照。"

### 痛点一览

| 痛点 | 具体表现 | 影响 |
|------|---------|------|
| **数据清洗耗时** | 缺失值、异常值、格式不统一 | 每次分析前必做 |
| **分析维度单一** | 只能做简单的同比环比 | 缺乏深度洞察 |
| **报表制作繁琐** | 每月重复制作相同格式报表 | 月月加班 |
| **可视化能力弱** | Excel图表不够专业 | 汇报不好看 |

## 6.2 用 WorkBuddy 做数据分析

### 方案一：快速数据清洗

把乱糟糟的数据扔给 WorkBuddy：

```
清洗这个 Excel 文件中的数据：

1. 删除空白行和重复行
2. 统一日期格式为 YYYY-MM-DD
3. 金额列去掉千分位逗号，保留2位小数
4. 找出并标注以下异常：
   - 金额为0或负数的行
   - 日期在未来的行
   - 必填字段为空的行

输出：清洗后的数据 + 问题清单
```

---

### 方案二：多维分析

有了干净数据，直接让 WorkBuddy 分析：

```
分析这个 Excel 文件中的财务数据：

分析维度：
1. 主要指标汇总（营收、成本、费用、利润）
2. 同比分析（本月 vs 去年同期）
3. 环比分析（本月 vs 上月）
4. 费用结构分析（各费用占比）
5. 异常标注（变动超过20%的项目）

输出：
- 汇总表格
- 3-5个关键发现
- 2-3条建议
```

### 方案三：自动生成分析报告

不想自己写报告？WorkBuddy 可以帮你：

```
基于以下 Q1 财务数据，生成一份分析报告：

数据：

- 营业成本：3,480 万元
- 销售费用：520 万元 | 管理费用：380 万元

报告结构：

三、变动分析（重点说明毛利率变化）
四、风险提示（标注 ⚠️ 的需重点关注项）
五、管理建议（3条具体可执行的建议）

要求：金额单位万元，全文不超过800字，适合管理层阅读
```

## 6.3 实战案例

### 案例：月度经营分析报告

**任务背景**：每月5号前要提交月度经营分析报告，内容包括：收入分析、成本分析、费用分析、利润分析。

**传统方式**：2天（收集数据 → 清洗 → 分析 → 写报告 → 做图表）

**用 WorkBuddy**：

```
第一步：数据清洗
"清洗桌面上的'3月财务数据.xlsx'，统一格式，标注异常数据"

第二步：快速分析
"分析这份数据，输出：
- 营收、成本、费用、利润汇总
- 各指标同比环比
- 费用占比
- 异常项标注"
（5分钟后得到分析结果）

第三步：生成报告
"基于以上分析结果，生成一份800字以内的月度经营分析报告，
包含执行摘要、核心指标、变动分析、风险提示、管理建议"
（3分钟后得到报告草稿）

第四步：准备图表数据
"准备3个图表的数据：
1. 营收趋势（近6个月）
2. 费用构成饼图
3. 同比对比柱状图"
（得到可以直接复制到PPT的数据）
```

**结果**：2小时完成2天的工作，有充足时间复核和优化。

## 6.4 常用分析模板

### 模板1：费用分析
```
分析这个 Excel 中的费用报销数据：

1. 按部门统计总费用和人均费用
2. 按费用类型统计占比
3. 计算环比增长率
4. 标注单笔超过5000元的大额支出

输出：部门排名表 + 类型占比表 + 大额支出清单
```

### 模板2：利润分析
```
分析这份利润表数据：

1. 计算毛利率、净利率、费用率
2. 与预算对比，计算完成率
3. 与去年同期对比，计算同比增长率
4. 找出变动最大的3个项目

输出：指标汇总表 + 对比分析 + 重点关注项
```

### 模板3：现金流分析
```
分析这个月的银行流水数据：

1. 汇总收入和支出总额
2. 按款项性质分类（货款、工资、费用、投资等）
3. 找出大额收支（超过10万的）
4. 计算月末余额

输出：收支汇总表 + 分类明细 + 余额核对
```

### 模板4：预算执行分析
```
对比实际数据与预算数据：

实际数据：[粘贴数据]
预算数据：[粘贴数据]

1. 计算各项目执行率
2. 找出超预算和低于预算的项目
3. 分析偏差原因
4. 预测全年完成情况

输出：执行率对比表 + 偏差分析 + 预测
```

## 6.5 常见问题

### Q1：WorkBuddy 处理数据量大时很慢？

解决方案：
- 可以先抽样测试（取前100行）
- 复杂分析可以分步做
- 大文件可以先分割

### Q2：分析结果有误差怎么办？

WorkBuddy 会尽力准确分析，但财务数据涉及精确计算：
- 关键数据要人工复核
- 可以在指令中要求标注置信度
- 发现错误直接让它重算

### Q3：报告生成太泛泛，不够具体？

提供更多背景信息：

```
补充背景：
- 我们是一家50人的互联网公司
- 本月营收增长主要来自老客户续费
- 成本增加是因为新租了办公室

请在报告中重点说明这些因素的影响
```
