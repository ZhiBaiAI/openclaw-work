## 4.1 WorkBuddy 的对话方式

### 和传统 AI 助手的区别

WorkBuddy 是一个**任务执行型 AI**，不像 ChatGPT 那样"能聊不能做"。

| 对比项 | 传统 AI 助手 | WorkBuddy |
|--------|-------------|-----------|
| **工作方式** | 你问，它答 | 你说，它做 |
| **文件处理** | 不能直接操作文件 | 可以读取、分析、整理文件 |
| **执行能力** | 只能给建议 | 能直接完成任务 |
| **对话风格** | 需要设定角色 | 直接说需求就行 |

### 和 WorkBuddy 说话的两种方式

**方式一：直接说你要什么**（最常用）

```
整理桌面上所有发票图片，按月份分类
```

**方式二：给任务 + 给数据 + 给格式**（更精准）

```
分析这个 Excel 文件（附件），计算每个部门的费用总额和占比，输出表格
```

## 4.2 好的财务指令长什么样？

### 好指令的两个特点

1. **说清楚要什么结果**
2. **说清楚输出格式**

### 三组对比案例

**对比一：报表分析**

| ❌ 模糊指令 | ✅ 清晰指令 |
|-----------|-----------|
| 帮我分析这个报表 | 分析附件中的 Q1 利润表，计算各项目同比变化，输出表格 |

---

**对比二：发票整理**

| ❌ 模糊指令 | ✅ 清晰指令 |
|-----------|-----------|
| 帮我看看这些发票 | 读取这5张发票图片，提取：发票号、日期、金额、开票单位，然后汇总成表格 |

---

**对比三：费用统计**

| ❌ 模糊指令 | ✅ 清晰指令 |
|-----------|-----------|
| 算一下花了多少钱 | 按费用类型汇总3月份的报销数据，输出：类型 | 金额 | 占比 表格 |

## 4.3 财务场景指令模板

以下是几个常用财务场景的直接可用的指令模板。

### 模板一：发票识别与整理

```
读取文件夹中所有发票图片，提取每张发票的：
- 发票号码
- 开票日期
- 购买方名称
- 销售方名称
- 金额（含税/不含税）
- 税额
- 发票类型

然后：
1. 按发票类型分类汇总金额
2. 标注金额最大的5张发票
3. 检查是否有信息不完整的发票
4. 输出汇总表格
```

**适用场景**：月末整理发票、发票归档

---

### 模板二：费用分析

```
分析这个 Excel 文件中的报销数据：

1. 按部门统计总费用和人均费用
2. 按费用类型统计，找出占比最大的三项
3. 计算环比增长率（本月 vs 上月）
4. 标注单笔超过5000元的支出

输出：
- 部门费用排名表
- 费用类型占比表
- 大额支出清单
```

**适用场景**：月度费用分析、预算执行分析

---

### 模板三：银行对账

```
对比以下两组数据：
- 银行对账单（附件A）
- 财务系统记录（附件B）

1. 找出完全匹配的交易
2. 找出不匹配的项，按类型分类：
   - 银行已收企业未收
   - 银行已付企业未付
   - 企业已收银行未收
   - 企业已付银行未付
3. 计算差异总金额
4. 输出未匹配项明细表
```

**适用场景**：月度对账、银行调节表编制

---

### 模板四：合同财务条款检查

```
审核以下合同中的财务条款：

1. 付款条件：预付比例是否合理，付款节点是否明确
2. 发票条款：开票时间、税率、含税/不含税约定
3. 违约条款：违约金计算方式是否对我方有利
4. 税务风险：税率适用是否正确，有无税务风险点

输出：
- 问题清单（高/中/低风险分级）
- 每项问题的原文 + 修改建议
- 必须关注的3个关键风险点
```

**适用场景**：合同审批、采购合规

---

### 模板五：财务报告生成

```
基于以下数据，生成一份 Q1 财务分析报告：

数据：
- 营收：5,230 万元（去年同期 4,180 万元）
- 营业成本：3,480 万元（去年同期 2,850 万元）
- 销售费用：520 万元
- 管理费用：380 万元
- 财务费用：85 万元
- 净利润：416 万元

报告结构：
一、执行摘要（3-5句话）
二、核心指标表（本期 | 同比 | 环比 | 趋势）
三、变动分析（重点说明毛利率变化）
四、风险提示
五、管理建议（3条）

要求：金额单位万元，保留1位小数，全文不超过800字
```

**适用场景**：月度/季度经营分析报告

---

### 模板六：数据清洗

```
清洗这个数据文件：

1. 删除空白行和重复行
2. 统一日期格式（YYYY-MM-DD）
3. 金额列：去掉千分位逗号，保留2位小数
4. 找出并标注以下异常：
   - 金额为0或负数的行
   - 日期在未来的行
   - 必填字段为空的行
5. 输出清洗后的数据 + 问题清单
```

**适用场景**：数据导入前的预处理

## 4.4 指令优化技巧

### 技巧一：指定输出格式

**问题**：AI 返回的结果格式不固定，你还要手动整理。

**解决方法**：明确告诉 AI 你要什么格式。

```
❌ 帮我分析一下这些数据

✅ 帮我分析这些数据，输出格式：
| 部门 | 1月 | 2月 | 3月 | 环比 |
|------|-----|-----|-----|------|
```

---

### 技巧二：限制输出范围

**问题**：AI 给的内容太多，超出你需要。

**解决方法**：告诉 AI 你只需要哪些。

```
❌ 介绍一下发票的类型和税务处理

✅ 介绍一下我们常见的3种发票类型，重点说明：
1. 增值税专用发票和普通发票的区别
2. 哪种情况下必须开专票
3. 开票时限要求
```

---

### 技巧三：一次只说一件事

**问题**：一次提太多要求，每个都做不好。

**解决方法**：分开多次说。

```
❌ 帮我分析报表、做预算、写报告、起草邮件

✅ 分开做：
1. "先帮我分析一下Q1的利润表"
2. "基于分析结果，帮我起草一份Q2预算建议"
3. "把这份预算建议整理成PPT大纲"
```

---

### 技巧四：提供参考上下文

**问题**：AI 不理解你的具体情况，给的建议不实用。

**解决方法**：提供必要的背景信息。

```
❌ 帮我写一份费用管控建议

✅ 帮我写一份费用管控建议
背景：
- 我们是一家50人的科技公司
- 3月份差旅费环比增长40%，达到28万
- 公司目前处于控制成本阶段
- 预算目标是全年费用零增长

请给出3条具体可执行的差旅费管控建议
```

## 4.5 常见问题处理

### Q1：WorkBuddy 没有按我说的做怎么办？

**检查清单**：
1. 指令是否具体？（"整理文件"太模糊 → "把桌面上的PDF文件移动到 D:\财务档案\2026年\"）
2. 是否有必要的权限？（文件夹是否已授权访问）
3. 文件路径是否正确？

**优化方法**：把指令拆得更细

```
❌ "帮我整理发票"

✅ "读取 D:\报销\2026年3月 文件夹中所有 .pdf 文件，
   提取每张发票的金额和日期，
   按日期重命名为 '发票_YYYYMMDD_金额.pdf' 格式，
   移动到 D:\报销\归档\2026年3月 文件夹"
```

---

### Q2：输出的数据有错误怎么办？

**方法一**：直接指出错误，让 AI 修正

```
你刚才计算的总金额是 28.5 万，但我检查发现应该是 28.3 万。
请重新核对这个数据，并检查其他计算是否正确。
```

**方法二**：提供更清晰的计算规则

```
重新计算时注意：
1. 金额单位是"元"，不是"万元"
2. 汇率统一用 7.25
3. 四舍五入保留2位小数
```

---

### Q3：AI 处理大文件时很慢怎么办？

WorkBuddy 处理大文件时：
- 可以分批处理
- 或者明确告诉 AI 只处理最近的数据

```
✅ "先分析3月份的数据，这个文件里有1-6月的数据，先只看3月"
```
