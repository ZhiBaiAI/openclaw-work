# 第9章：高级篇 — 多任务协同

> **预期效果**：🚀 同时处理多个任务，大幅提升效率

---

## 9.1 什么是多任务协同？

### 从一个任务到多个任务

**单个任务**：
```
输入：帮我分析3月费用
输出：分析结果
```

**多任务协同**：
```
输入：同时处理以下工作：
1. 分析3月费用
2. 整理发票
3. 生成周报

WorkBuddy 自动分配处理：
- 任务1：分析费用（进行中）
- 任务2：整理发票（进行中）
- 任务3：生成周报（进行中）

输出：三个任务的结果汇总
```

---

## 9.2 如何发起多任务

### 方法一：一次说多个任务

```
同时处理以下工作：
1. 分析这个 Excel 文件的营收数据
2. 检查这封邮件中的发票是否合规
3. 整理桌面上的报销文件

完成后汇总结果
```

### 方法二：并行处理

```
同时执行以下任务（可以并行处理）：
- 任务A：读取 D:\数据\客户表.xlsx，统计客户数量
- 任务B：读取 D:\数据\产品表.xlsx，统计产品数量
- 任务C：读取 D:\数据\订单表.xlsx，统计订单总额

输出三份统计结果
```

### 方法三：链式任务

```
第一步：读取发票文件夹，统计发票数量和总金额
第二步：基于第一步的结果，生成费用分析报告
第三步：将报告保存到桌面

完成后告诉我总金额是多少
```

## 9.3 财务场景实战

### 场景一：月末大扫除

月末需要同时处理多项工作：

```
同时执行以下月末工作：

任务1：发票整理
- 读取桌面"待整理"文件夹中的所有发票
- 按月份和类型分类
- 生成汇总表

任务2：银行对账
- 对比银行对账单和账套记录
- 生成调节表

任务3：数据备份
- 将本月所有财务文件备份到 D:\备份\YYYYMM
- 生成备份清单

完成后发送处理汇总
```

### 场景二：周会更准备

周会前需要准备多份材料：

```
准备周会材料，同时处理：

材料1：本周报销汇总
- 读取本周报销数据
- 按部门统计
- 生成汇总表

材料2：本周销售数据
- 读取销售数据
- 计算同比环比
- 生成简报

材料3：本周问题清单
- 汇总本周需要跟进的问题
- 按优先级排序

完成后整合成一份周会材料包
```

### 场景三：季度报告准备

季度末需要全面分析：

```
准备季度报告，同时进行以下分析：

分析1：营收分析
- 分析Q1营收数据
- 按产品/客户/地区分类
- 计算同比增长

分析2：成本分析
- 分析Q1成本数据
- 找出成本变动原因
- 与预算对比

分析3：费用分析
- 分析Q1费用数据
- 各部门费用占比
- 超预算项标注

分析4：利润分析
- 计算毛利率、净利率
- 同比环比分析
- 趋势判断

汇总所有分析结果，生成季度报告草稿
```

## 9.4 任务队列管理

### 查看进行中的任务

```
显示当前进行中的任务
```

### 调整任务优先级

```
当前有以下任务：
1. 发票整理（进行中）
2. 数据分析（等待中）
3. 生成周报（等待中）

将"生成周报"优先级调整为最高
```

### 取消不需要的任务

```
取消"数据分析"这个任务
```

### 等待任务完成

```
继续等待当前任务完成，然后执行：
汇总所有结果，生成最终报告
```

## 9.5 实战案例

### 案例：财务经理的一天

**背景**：张经理是公司财务经理，每天需要处理多项工作：

**早上9点：一键启动今日工作**
```
同时处理：
1. 检查银行对账单是否有新交易
2. 查看本月报销审批待处理数量
3. 检查本月预算执行情况
4. 汇总昨日重要财务数据

完成后发送汇总给我
```

**结果**：5分钟得到所有数据，而不是花1小时逐项检查

---

**上午10点：处理待办事项**
```
同时处理以下待办：
1. 审核3笔大额报销（>1万）
2. 跟进2个客户的对账确认
3. 更新本月的预算执行表
4. 准备下午会议的材料

完成后标记已完成的，列出未完成的
```

**结果**：并行处理，30分钟完成原本需要2小时的工作

---

**下午4点：生成日报**
```
汇总今日工作：
1. 今日审批的报销数量和金额
2. 今日处理的银行交易数量
3. 今日完成的对账单
4. 需要跟进的问题

生成今日工作日报，保存到桌面
```
