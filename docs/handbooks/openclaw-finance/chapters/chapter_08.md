# 第8章：进阶篇 — 搭建专属财务工作流

> **预期效果**：🎯 让 WorkBuddy 记住你的工作习惯，每次一说就执行

---

## 8.1 什么是工作流？

### 工作流 vs 一次对话

| 对比 | 一次对话 | 工作流 |
|------|---------|--------|
| **使用方式** | 每次都要说清楚要什么 | 一次配置，长期使用 |
| **适用范围** | 临时任务 | 重复性任务 |
| **效率** | 每次重新描述 | 一句话搞定 |
| **记忆能力** | 对话结束就忘 | 记住你的习惯 |

### 举个例子

**一次对话**：
```
你：帮我分析3月的费用
WorkBuddy：好的，请提供数据
你：[粘贴数据]
WorkBuddy：[分析结果]
（下次还要重来一遍）
```

**工作流**：
```
你：执行月度费用分析
WorkBuddy：✅ 已执行
- 自动读取3月数据
- 完成多维分析
- 生成报告
（以后每月只需说这一句话）
```

## 8.2 创建你的第一个工作流

### 方法：直接告诉 WorkBuddy

WorkBuddy 支持"教会它做事"：

```
请记住以下工作流程：

【工作流名称】
月末发票汇总

【执行步骤】
1. 读取桌面上"3月报销"文件夹中的所有发票文件
2. 提取每张发票的金额、日期、销售方
3. 按费用类型分类（差旅费、办公费、招待费、其他）
4. 生成汇总表，保存到桌面

以后我输入"执行月末发票汇总"时，自动执行这个流程
```

### 保存工作流

WorkBuddy 会自动保存你教给它的流程。以后每次说关键词，就能自动执行。

## 8.3 财务常用工作流模板

### 模板1：月度对账工作流

```
请记住这个工作流：

【工作流名称】
银行对账

【触发关键词】
对账、银行核对

【执行步骤】
1. 读取"银行对账单.xlsx"
2. 读取"账套记录.xlsx"
3. 对比两份数据，找出未匹配项
4. 按类型分类差异项
5. 生成银行余额调节表
6. 将结果保存为"对账结果_YYYYMMDD.xlsx"

【输出】
- 未匹配项明细表
- 银行余额调节表
- 差异金额汇总
```

---

### 模板2：发票归档工作流

```
请记住这个工作流：

【工作流名称】
发票归档

【触发关键词】
归档发票、整理发票

【执行步骤】
1. 扫描桌面"待归档"文件夹中的发票文件
2. 读取每张发票的日期和金额
3. 按日期和类型重命名文件（格式：类型_YYYYMMDD_金额.pdf）
4. 按月份整理到对应文件夹
5. 生成归档清单

【文件夹结构】
D:\发票归档\
├── 2026年/
│   ├── 01月/
│   ├── 02月/
│   └── 03月/
```

---

### 模板3：周报生成工作流

```
请记住这个工作流：

【工作流名称】
生成周报

【触发关键词】
生成周报、写周报

【执行步骤】
1. 读取本周工作记录文件（如果有）
2. 读取本周的报销数据、销售数据（如果有）
3. 汇总本周主要工作
4. 生成周报，格式：
   - 本周完成工作
   - 下周计划
   - 问题与风险
   - 数据摘要
5. 保存到桌面，文件名"周报_YYYYMMDD.md"
```

---

### 模板4：报表分析工作流

```
请记住这个工作流：

【工作流名称】
财务分析

【触发关键词】
分析报表、分析财务

【执行步骤】
1. 读取指定的 Excel 文件
2. 计算核心指标：营收、成本、费用、利润
3. 计算同比、环比增长率
4. 分析费用结构
5. 标注异常项（变动超过20%）
6. 生成分析报告

【报告格式】
- 执行摘要
- 核心指标表
- 变动分析
- 风险提示
- 建议
```

## 8.4 工作流高级设置

### 设置定时执行

有些工作可以定时自动执行：

```
设置定时任务：
- 名称：月度对账提醒
- 时间：每月5号早上9点
- 执行：检查是否有未完成的银行对账
- 提醒方式：弹出通知
```

### 设置触发条件

```
设置触发条件：
- 监控文件夹：D:\报销\待处理
- 触发条件：文件夹中有新文件
- 自动执行：读取发票 → 提取信息 → 汇总
- 完成后通知我
```

### 设置审批流程

```
设置审批流程：
- 工作流：发票审核
- 第一步：提取发票信息
- 第二步：检查金额是否超过5000元
  - 不超过：自动通过
  - 超过：发送给我审批
- 第三步：审批通过后归档
```

## 8.5 实战案例

### 案例：财务小李的自动化

**背景**：财务小李每月重复做以下工作：
- 每周一：汇总上周报销数据
- 每月5号：完成银行对账
- 每月10号：生成月度分析报告

**小李的做法**：

```
第一步：教会 WorkBuddy 周报流程
[输入模板，WorkBuddy 保存]

第二步：教会 WorkBuddy 对账流程
[输入模板，WorkBuddy 保存]

第三步：教会 WorkBuddy 月度分析流程
[输入模板，WorkBuddy 保存]

第四步：设置提醒
- 每周一早上9点：提醒"执行周报"
- 每月5号早上9点：提醒"执行对账"
```

**结果**：
- 以前：每月花在重复工作上的时间约20小时
- 现在：只需确认执行结果，有问题再处理
- 节省时间：约15小时/月
