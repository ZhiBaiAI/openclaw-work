## 2.1 为什么选择 WorkBuddy？

在开始安装之前，先简单说说为什么这本手册推荐 WorkBuddy 而不是其他工具。

### WorkBuddy 是什么？

WorkBuddy 是腾讯推出的**桌面端 AI 助手**。你可以把它理解成一个"长在电脑里的 AI 同事"——你告诉它要做什么，它真的会在你的电脑上帮你完成。

### 和其他 AI 工具的区别

| 工具 | 能做什么 | 财务场景 |
|------|---------|---------|
| **ChatGPT / Kimi** | 你问，它答 | ❌ 只能聊天，不能操作文件 |
| **WorkBuddy** | 你说，它做 | ✅ 直接帮你整理发票、生成报表 |

### WorkBuddy 对财务人员的优势

- **开箱即用** — 不需要配置复杂的环境
- **腾讯背书** — 账号安全有保障，企业采购没顾虑
- **原生文件处理** — Excel、发票、PDF 直接操作
- **财务友好** — 专注办公场景，不需要写代码

## 2.2 安装前的准备

### 电脑配置要求

WorkBuddy 对电脑要求不高，普通的办公电脑都能跑：

| 配置 | 要求 |
|------|------|
| **操作系统** | Windows 10/11 或 macOS 11+ |

### 需要提前安装的软件（如果有就不用装）

WorkBuddy 运行需要几个基础软件。**大多数电脑已经有了**，你可以先试试安装 WorkBuddy，如果提示缺软件再装。

**需要但可能没有的软件：**

| 软件 | 作用 | 大多数人有吗 |
|------|------|-------------|
| **Node.js** | 运行环境 | ⚠️ 程序员电脑通常有 |
| **.NET 桌面运行时** | 系统交互 | ✅ Windows 电脑通常有 |

## 2.3 安装 WorkBuddy（Windows 系统）

### 第一步：下载安装包

1. 打开浏览器，访问：**https://www.codebuddy.cn/fission/?inviteCode=2n21s6sx2zu7k17b**，注册即得3000虾米

2. 点击"下载安装包"按钮

3. 选择 Windows 版本下载

> 💡 **提示**：下载的文件是一个 `.exe` 安装程序，大小约 100-200MB。

### 第二步：运行安装程序

1. 找到下载的文件（通常在"下载"文件夹），双击运行

2. 如果弹出"Windows 保护你的电脑"，点击"仍要运行"

3. 安装向导会打开，跟着提示走：
   - 点击"下一步"
   - 选择安装位置（默认位置就可以）
   - 点击"安装"
   - 等待安装完成

4. 安装完成后，勾选"启动 WorkBuddy"，点击"完成"

### 第三步：登录账号

1. WorkBuddy 启动后会弹出登录界面

2. 你可以使用以下方式登录：
   - **微信扫码**（推荐，最快）
   - **手机号验证码**
   - **腾讯云账号**

3. 登录成功后，你就进入了 WorkBuddy 主界面

### 第四步：绑定微信（可选但推荐）

绑定微信后，你可以在手机微信里发送指令，操控电脑上的 WorkBuddy 帮你工作。

**绑定步骤**：

1. 打开 WorkBuddy，点击右上角头像 → **设置**
2. 找到「**微信客服号集成**」，扫码绑定
3. 绑定成功后，在微信里给 "WorkBuddy" 发消息即可

**使用示例**：
- 在微信里发一张发票图片 → 自动识别发票信息
- 在微信里发 "整理本月报销" → 自动处理 Excel 文件
- 在微信里发 "生成月度报告" → 自动生成分析报表

> 💡 **注意**：电脑需要开机且 WorkBuddy 在运行，微信里发指令才能生效。

### 第五步：验证安装成功

安装完成后，你应该能看到类似这样的界面：

1. 桌面上出现了 WorkBuddy 的图标
2. 点击图标打开，看到主界面
3. 底部有一个对话框，可以输入文字

**恭喜！安装成功了！**

## 2.7 常见问题解答

### Q1：下载很慢怎么办？

可以尝试：
- 使用手机热点下载
- 换一个网络环境（公司网络可能限速）

### Q2：安装时提示"无法安装"？

解决方案：
1. 确保关闭了杀毒软件
2. 右键安装文件，选择"以管理员身份运行"
3. 如果还是不行，试试重启电脑后再安装

### Q3：登录时收不到验证码？

- 检查手机信号
- 查看拦截短信
- 尝试用微信扫码登录

### Q4：WorkBuddy 运行时很卡？

可能原因：
- 内存不够：关闭其他程序
- 同时运行太多任务：等待之前的任务完成

### Q5：任务执行失败了怎么办？

WorkBuddy 会显示具体的错误原因：
- 如果是"文件找不到"，检查文件路径是否正确
- 如果是"权限不足"，重新授权文件夹访问

### Q6：怎么让 WorkBuddy 只处理特定文件夹？

可以在设置中限制 AI 只能访问指定的文件夹：
1. 点击右上角"设置"
2. 选择"权限管理"
3. 设置"允许访问的文件夹"

## 2.9 WorkBuddy 能做什么 / 不能做什么

**了解边界，才能用好 AI。**

### ✅ WorkBuddy 擅长的

| 场景 | 说明 | 效率提升 |
|------|------|---------|
| **发票识别** | 读取发票图片，提取金额、日期、税号等信息 | 30倍+ |
| **数据整理** | Excel分类、汇总、去重、格式转换 | 50倍+ |
| **报表分析** | 读取财务数据，生成摘要、图表、趋势分析 | 10倍+ |
| **文件处理** | 批量重命名、移动、整理归档 | 20倍+ |
| **文本生成** | 写财务报告初稿、邮件、说明文档 | 5倍+ |
| **数据核对** | 检查两表差异、发现异常数据 | 10倍+ |

### ❌ WorkBuddy 不擅长的

| 场景 | 原因 | 建议 |
|------|------|------|
| **直接登录系统操作** | 无法直接登录网银/ERP等系统 | 需要导出数据后再处理 |
| **精确数学计算** | AI计算可能有小数误差 | 重要数字用Excel公式复核 |
| **实时数据查询** | 无法连接公司内部数据库 | 需要导出数据 |
| **判断涉税风险** | 涉及专业判断，需人工确认 | AI整理数据，人做判断 |
| **实时获取政策变化** | 知识有截止日期 | 关注官方渠道，及时更新 |

### 💡 正确的使用心态

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   WorkBuddy = 超级助理                               │
│                                                     │
│   你做决策，它干活                                    │
│   你来复核，它来整理                                  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**核心原则**：AI 做 80% 的重复劳动，你做 20% 的关键判断。
